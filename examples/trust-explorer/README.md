# Trust Explorer

Explore Circles trust relationships - see who trusts you, who you trust, and discover trust networks across the ecosystem.

## Features

- View your incoming and outgoing trust relationships
- Search for any address or name to explore their trust network
- Click through profiles to navigate the trust graph
- Mobile responsive

## Development

```bash
npm install
npm run dev
```

## Deployment

```bash
npm run build
vercel --prod
```

Remember to disable Vercel Deployment Protection after deploying.

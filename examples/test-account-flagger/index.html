How it works

  • Add any addresses you want to flag — test accounts, bots, old/unused accounts
  • The list is stored as ##TEST_ACCOUNTS## in your Circles profile metadata
  • Click "Save to Profile" to pin to IPFS and update on-chain (requires a signed transaction)
  • Analytics dashboards and TMSs aggregate flagged addresses across all profiles
  • You can add or remove addresses at any time — each save is one on-chain tx (gas: fractions of a cent on Gnosis Chain)
